Student Performance Data Analysis

Project Overview
This project focuses on analyzing student academic performance using Python and data analysis libraries. The dataset used is `student-mat.csv`, which contains information about students, including study time, grades, gender, and other academic-related factors.

The main objective of this project is to explore, clean, analyze, and visualize the dataset to gain insights into student performance patterns.

 Objectives
- Load and explore the student performance dataset.
- Check missing values and clean the dataset.
- Remove duplicate records.
- Analyze student final grades (`G3`).
- Identify high-scoring students.
- Study the relationship between study time and academic performance.
- Compare male and female student performance.
- Create visualizations for better understanding.

 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab (Jupyter Notebook)

 Dataset Information
Dataset Name: **student-mat.csv**

The dataset contains information about:
- Student demographics
- Study habits
- Academic grades
- Attendance and performance-related factors

Data Cleaning Performed
- Checked missing values using `.isnull()`
- Removed duplicate rows using `drop_duplicates()`
- Inspected dataset shape and data types

Analysis Questions
The following analysis questions were answered:

1. What is the average final grade (`G3`)?
2. How many students scored above 15?
3. Is study time correlated with student performance?
4. Which gender performs better on average?

Visualizations
The following visualizations were created:
- Histogram of final grades
- Scatterplot of study time vs grades
- Bar chart comparing male vs female average scores

Conclusion
This project helped in understanding student academic performance through data analysis and visualization. Study time, grades, and gender-wise performance were analyzed to identify meaningful patterns and trends in the dataset.
